# todo.py
This project is a Command-Line To-Do List Application in Python that allows users to manage daily tasks efficiently. Users can add tasks, view tasks, mark tasks as completed, and delete tasks using a simple menu-driven interface. The application runs continuously until the user chooses to exit, making it ideal for beginners to understand .
